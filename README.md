# r-tree-rs

R-tree spatial indexing with R*-tree splits in pure Rust.

## Features

- **Insertion** — With R*-tree choose subtree and split heuristics
- **Deletion** — Remove entries with reinsertion
- **Range Query** — Axis-aligned bounding box queries
- **k-NN** — k-nearest neighbor spatial search

No external dependencies. Pure `std` Rust.
